容器方式部署

容器部署方式是社区朋友贡献的，欢迎大家试用
  1. 下载夜莺repo的内容
  2. cd dockerfiles
  3. docker-compose up
  4. 修改hosts 加入 127.0.0.1 n9e.example.com
  5. 浏览器访问 http://n9e.example.com
  6. 用户名 root 密码 root.2020
